About

This is a stock exchange website.

Users can view the chart and other data for all the companies listed on the stock exchange.

Users can also create portfolios to track the performance of groups of companies.

Company Listing Page

The Company Listing Page lists all the companies according to their company symbol in alphabetical order.

Clicking on a specific letter will show a list of all the companies that have a company symbol starting with that letter.

Clicking on one of the listed companies will take the user to the Company Profile Page for that company.


On the navigation bar, click on "Stocks" to go to the Company Listing Page.

Company Profile Page

The Company Profile Page shows the quote data and the chart data for a specific company.

The quote data for a company includes the current price, the last close price, and how much the price has changed since last close.

The chart data for a company includes the stock price for the past 1 year. The user can also choose to only view the stock price for the past 6 months, or the past 3 months.


On the navigation bar, click on "Stocks" to go to the Company Listing Page. Click on one of the listed companies to go to its Company Profile Page.

Portfolio Listing Page

The Portfolio Listing Page lists all the portfolios that have been created by a specific user.

Clicking on a specific letter will show a list of all the portfolios that have a portfolio name starting with that letter.

Clicking on one of the listed portfolios will take the user to the Portfolio Profile Page for that portfolio.


The "Create Portfolio" button can be used to create a new portfolio.

The "Delete Portfolio" button can be used to delete an existing portfolio.


On the navigation bar, click on "Portfolios" to go to the Portfolio Listing Page.

Portfolio Profile Page

The Portfolio Profile Page shows the investment value data and the chart data for a specific portfolio.

A portfolio is composed of investments in a group of companies. The investment value of the portfolio is tracked from the "Invest Start Day" to today.

The investment value data for a portfolio includes the value of the portfolio at the beginning of the investment period, the value of the portfolio at the end of the investment period, and the change in the value of the portfolio over the investment period.

The chart data for a portfolio includes the value of the portfolio for the past 1 year. The user can also choose to only view the value of the portfolio for the past 6 months, or the past 3 months.


The "Portfolio Settings" button can be used to update the settings for the portfolio.

The "Add Investment" button can be used to add an investment in a specific company.

The "Remove Investment" button can be used to remove an investment in a specific company.


On the navigation bar, click on "Portfolios" to go to the Portfolio Listing Page. Click on one of the listed portfolios to go to its Portfolio Profile Page.
